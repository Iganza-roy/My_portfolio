AiBnB clone

AirBnB clone

This project is an AirBnB clone that began with the creation of a command-line interpreter (console) to manage instances of various classes. The application includes models for User, Place, State, City, Amenity, and Review, each inheriting from a base model.

In the updated version, a user-friendly interface was introduced for enhanced interactivity. The project was built using Python for the console, MySQL and JSON for storage, Flask for API interaction and as the framework, and HTML, CSS, and JavaScript for the frontend.

Simple Shell

Simple Shell

This project is a custom shell built in C, designed to display a prompt and wait for user commands. Each command line ends with a new line, and the prompt reappears after a command is executed.

The shell handles simple, single-word command lines without semicolons, pipes, redirections, or advanced features. No arguments are passed to programs. If an executable cannot be found, an error message is printed, and the prompt is displayed again. The shell is capable of handling errors effectively.
